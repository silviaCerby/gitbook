# Edit an app setting restriction

{% hint style="info" %}


**Who can use this feature?**

* Workspace **Owners** , **Super Admins** , and **Admins**
* Only supported using the Cerby web app
* Workspace users must have the Cerby browser extension installed and logged in


{% endhint %}

As a workspace **Owner** , **Super Admin** , or **Admin** , you can edit existing [app setting restrictions](https://help.cerby.com/en/articles/10276184-explore-app-settings-restrictions) so that you add, remove, or configure these restrictions.

To edit an app settings restriction, you must complete the following steps:

1. Log in to your [Cerby](https://app.cerby.com/) workspace using the Cerby web app.
2. Select the **Settings** option from the left navigation drawer. The **Workspace Configuration** page is displayed with the **General** tab activated.
3. Activate the **Extension settings** tab.
4. Activate the left **App settings restrictions** tab.
5. Click the **More options**() button on the right side of the app you want to edit. A drop-down menu is displayed.
6. Click the **Edit restrictions** option. A dialog box is displayed with the existing restrictions configured for the app.
7. Modify the restrictions for the app as needed.
**NOTE:** To turn off all restrictions for the setting in the app, deactivate the switches in the main section.

8. Click the **Apply restrictions** button. The dialog box closes, and a success message box is displayed.

{% hint style="danger" %}


**IMPORTANT:** Restrictions take five minutes to be applied in the app.


{% endhint %}

Now you are done.
