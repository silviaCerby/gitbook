# Retrieve the entitlement string value from your IdLCM integration

{% hint style="info" %}


**Who can use this feature?**

* Workspace **Owners** , **Super** **Admins** , and **Admins**
* Only supported using the Cerby web app


{% endhint %}

As a workspace **Owner** , **Super** **Admin** , or **Admin** , you can retrieve the entitlement string value from your IdLCM integration. These values identify specific entitlements in your external application and indicate their associated access levels.

After connecting and configuring an integration to your IdP, you can centralize entitlement management in your IdP. When assigning entitlements to users, you must provide the corresponding string value to ensure each user is assigned to the correct entitlement and receives the appropriate access level.

To retrieve the entitlement string value from your IdLCM integration, you must complete the following steps:

1. Log in to your [Cerby](https://app.cerby.com/) workspace.
2. Select the **Integrations** option from the left menu. The **Integrations** page is displayed.
3. Click the **More options** (<img src="https://downloads.intercomcdn.com/i/o/pc0ldyqu/1751746198/55d5851ce30117bb19271d941e25/be3fdebe-e155-4174-aa73-db89244fe687?expires=1761858000&signature=25020b1178f9a4aaf30456e12ac5cc72bc18dd16d96d3c8fabfef1f943a1a10b&req=dSciF856m4BWUfMW3Hu4gf1t4cWU7UvUS%2Fi%2B1pIMvsPy18pwzKf%2Fmb5%2Bw%2FKA%0Arw%3D%3D%0A" alt="">) icon of the corresponding integration. A drop-down menu is displayed.
4. Select the **Settings** option from the menu. The integration details page is displayed with the **Settings** tab active.
5. Activate the **Entitlements** tab.
6. Click the **More options** (<img src="https://downloads.intercomcdn.com/i/o/pc0ldyqu/1751746198/55d5851ce30117bb19271d941e25/be3fdebe-e155-4174-aa73-db89244fe687?expires=1761858000&signature=25020b1178f9a4aaf30456e12ac5cc72bc18dd16d96d3c8fabfef1f943a1a10b&req=dSciF856m4BWUfMW3Hu4gf1t4cWU7UvUS%2Fi%2B1pIMvsPy18pwzKf%2Fmb5%2Bw%2FKA%0Arw%3D%3D%0A" alt="">) icon of the corresponding entitlement. A drop-down menu is displayed.
7. Select the **Value** option. A side drawer is displayed with the **Value** tab active.
Under the **Value string** section, you can view the list of available string values.

8. Click the **Copy** (<img src="https://downloads.intercomcdn.com/i/o/pc0ldyqu/1802982366/4f923229d1dfcca3229db300f637/AD_4nXctY-gr-ukLTXT0-SF0RxUP_RHj8k_50y1FfFQWuCarXgURXnDJs5RYwZwFM6cs0CskJrYE-RD_Oe7M6WKywof2oGZjaaCL2OFsolxKzFF01UFF2TLs62z64pC5n5qBwWXPhz0mxQ?expires=1761751350&signature=f1d103cb731ad31d77a71fdcd8cf6db8aca91a73244932c01a809a8db89dd79e&req=dSgnFMB2n4JZX%2FMU3HP0gN01tVjD9HSYVlLSWGZHDaaPNre0YhY%3D%0A" alt="">) icon of the role you want to grant to the users assigned to this entitlement.

Now you are done.
