# Remove an account from a collection

{% hint style="info" %}


**Who can use this feature?**

* Workspace**Owners** , **Super Admins** , **Admins** , **Users** , and **Guest Users**
* Collection **Owners**
* Only supported using the Cerby web app


{% endhint %}

As a collection **Owner** , you can remove an account from a collection. With this action, all workspace members and teams who have shared access to the collection automatically lose access to the account through the collection.

To remove an account from a collection, you must complete the following steps using the Cerby web app:

  1. Log in to your [Cerby](https://app.cerby.com/) workspace.
  2. Select the **Collections** option from the left menu. The **Collections** page is displayed.
  3.
