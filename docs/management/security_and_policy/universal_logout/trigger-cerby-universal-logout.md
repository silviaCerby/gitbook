# Trigger Cerby Universal Logout

{% hint style="info" %}


**Who can use this feature?**

  * Workspace**Owners** , **Super Admins** , and **Admins**
  * Only supported using the Cerby web app


{% endhint %}

As a workspace **Admin** , **Super** **Admin** , or **Owner** , you can use Cerby Universal Logout to terminate user access across all devices for the accounts assigned to a user in Cerby.

{% hint style="info" %}


**NOTE:** This version of Universal Logout is triggered solely from Cerby's dashboard.


{% endhint %}

To trigger Universal Logout in Cerby, you must complete the following steps:

  1. Log in to your [Cerby](https://app.cerby.com/) workspace.
  2. Select the **All members** option from the left navigation drawer. The **All Members** view is displayed with the **Members** tab activated.
  3. Search for the member whose sessions you want to terminate.
  4. Now you are done.
