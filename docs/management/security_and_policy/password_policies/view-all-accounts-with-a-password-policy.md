# View all accounts with a password policy

{% hint style="info" %}


**Who can use this feature?**

* Workspace**Owners** , **Super Admins** , and **Admins**
* Only supported using the Cerby web app


{% endhint %}

As a workspace **Admin** , you can view all the accounts belonging to a specific managed app for which a [password policy](https://cerby-test.gitbook.io/cerby-test/support-and-use-cases/explore/explore-password-policies) was created. To do so, you must complete the following steps:

  1. Log in to your [Cerby](https://app.cerby.com/) workspace.
  2. Select the **Password Policies** option from the left menu. The **Password Policies** page is displayed with a table of policies.
  3. Click the **View all accounts** (<figure><img src="https://downloads.intercomcdn.com/i/o/pc0ldyqu/1543939804/59dc940e9735b896797d94f9bab7/AD_4nXcPIqP-56DNIhh9Dn7gqcWeTHtmBKufYHqQs4op-rCPHVmeM0ilkjUpG1Fiotv8JLhThVFCrt5O8l130q7prKRoNerp750jxv31sgznh2PcxUGd9cMZf1uxI-oGZXXozUELbw7bkQ?expires=1748389050&signature=01caa5bfb91485400ecb40f001dd163f91f881e0408a5830f40185c48e9d704c&req=dSUjFcB9lIlfXfMU3HP0gJmgheHdXnkWf04cNnAReo%2FkG6YkcbI%3D%0A" alt=""><figcaption></figcaption></figure>) icon of the corresponding policy. The policy details page is displayed with a table of accounts affected by the policy.
**TIP:** You can also select the **View all accounts** option from the **More options**(<figure><img src="https://downloads.intercomcdn.com/i/o/pc0ldyqu/1543940067/1e0eda2076ca14c71147a80649d9/AD_4nXeT3knlE0wH2YpVbRA63-2VFqLllGBNGN5SpUNwPQUXio36aYQKc2xR73Rsshzru2iPd6vuBnUTPsDTHRpe-ZAD3ZEt1dkv4LptnA3tVSjjY5spE71WcvrKTN_ad8tMAiFwjAlaDw?expires=1748389050&signature=2a6f40b8899a846bc7c7723bca158eabca60d417cebeaed3d17f5bfb79170ea9&req=dSUjFcB6nYFZXvMU3HP0gJbHbB0WXsXNM%2BXDSr%2F6d66pYR894WI%3D%0A" alt=""><figcaption></figcaption></figure>) drop-down menu.

Now you are done.
