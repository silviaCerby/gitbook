# Exclude an account from a password policy

{% hint style="info" %}


**Who can use this feature?**

  * Workspace**Owners** , **Super Admins** , and **Admins**
  * Only supported using the Cerby web app


{% endhint %}

As a workspace **Admin** , you can exclude an account from an existing [password policy](https://help.cerby.com/en/articles/11465716-explore-password-policies) so that automated password rotations are not executed for it.

{% hint style="info" %}


**NOTE:** Already excluded accounts are identified with the **Excluded** status in the table of policies.


{% endhint %}

To exclude accounts from a password policy, you must complete the following steps:

  1. Log in to your [Cerby](https://app.cerby.com/) workspace.
  2. Select the **Password Policies** option from the left menu. The **Password Policies** page is displayed with a table of policies.
  3. The policy details page is displayed with a table of accounts affected by the policy.
**TIP:** You can also select the **View all accounts** option from the **More options**(<figure><img src="https://downloads.intercomcdn.com/i/o/pc0ldyqu/1543948265/7779c4da24f86ae23b910fd83f98/AD_4nXeT3knlE0wH2YpVbRA63-2VFqLllGBNGN5SpUNwPQUXio36aYQKc2xR73Rsshzru2iPd6vuBnUTPsDTHRpe-ZAD3ZEt1dkv4LptnA3tVSjjY5spE71WcvrKTN_ad8tMAiFwjAlaDw?expires=1748389650&signature=ef0eeb5f10b68f20e0ec9d6734a8680bef16694570c5acebc81b17357f16c7dc&req=dSUjFcB6lYNZXPMU3HP0gHa7hViZHLsQ20CF6JXqKMx%2Bp0MJrng%3D%0A" alt=""><figcaption></figcaption></figure>) drop-down menu.

  4. Select the **Exclude from policy** option from the **More options**(<figure><img src="https://downloads.intercomcdn.com/i/o/pc0ldyqu/1543948265/7779c4da24f86ae23b910fd83f98/AD_4nXeT3knlE0wH2YpVbRA63-2VFqLllGBNGN5SpUNwPQUXio36aYQKc2xR73Rsshzru2iPd6vuBnUTPsDTHRpe-ZAD3ZEt1dkv4LptnA3tVSjjY5spE71WcvrKTN_ad8tMAiFwjAlaDw?expires=1748389650&signature=ef0eeb5f10b68f20e0ec9d6734a8680bef16694570c5acebc81b17357f16c7dc&req=dSUjFcB6lYNZXPMU3HP0gHa7hViZHLsQ20CF6JXqKMx%2Bp0MJrng%3D%0A" alt=""><figcaption></figcaption></figure>) drop-down menu of the corresponding policy. The **Exclude account from policy?** dialog box is displayed.
  5. Click the **Exclude** button. The dialog box closes, and a success message box is displayed.

Now you are done.
