# [UNPUBLISHED] Install the Scout browser extension

{% hint style="info" %}


**Who can use this feature?**

* Cerby workspace**Owners** , **Super Admins** , and **Admins**
* Only supported using the Scout browser extension


{% endhint %}

As a workspace **Owner** , **Super Admin** , or **Admin** , you can install the Scout browser extension to scout the ways you complete user and account tasks on your apps and help accelerate the process of building the automation jobs you use in Cerby. As a workspace **user** , you must request authorization to install Scout from a Cerby Admin.

**IMPORTANT:** Your organization must request Cerby access to this extension because it is privately listed in the Chrome Web Store.
---

* * *

## Supported browsers

The Scout browser extension is supported only on the Chrome browser.

* * *

## Install Scout

To install Scout, you must complete the next steps:

  1. Navigate to the [Scout by Cerby](https://chromewebstore.google.com/detail/scout-by-cerby/dknnopjbgccecedcmkfjlmenbagocmnh?authuser=1) private listing in the Chrome Web Store.
  2. Click the **Add to Chrome** button. A confirmation dialog box is displayed with the list of extension permissions.
  3. Click the **Add extension** button to confirm the installation. The **Scout** icon () appears in the toolbar of your web browser.

Now you are done.
