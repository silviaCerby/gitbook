# Set up your personal vault

{% hint style="info" %}


**Who can use this feature?**

* Workspace **Owners** , **Super Admins** , **Admins,** and**Users**
* Only supported using the Cerby web app


{% endhint %}

As a workspace **Owner** , **Super Admin** , **Admin,** or**User** , you can set up a [personal vault](https://help.cerby.com/en/articles/13188897-explore-personal-vaults) to securely store your personal accounts in Cerby. You must complete this setup after a workspace **Admin** turns on the personal vault for you.

To set up your personal vault, you must complete the following steps:

1. Log in to your[ Cerby](https://app.cerby.com/) workspace.
2. Click your user profile located at the top right of the Cerby dashboard. A drop-down menu is displayed.
3. Select the **My Profile** option. The **My Profile** page is displayed with a banner indicating that a personal vault has been turned on for you.
4. Click the **Get started** button. A wizard is displayed on the **Welcome to Personal Vaults!** page.
5. Click the **Set up Personal Vault** button. The **Page reload required** dialog box is displayed on the wizard.
6. Click the **Reload page** button. The wizard closes, and a success message box is displayed.

Now you are done.
